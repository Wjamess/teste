resumo das tags para tabela, "simples"

  • table = cria uma tabela
  • tr = table row, ou linha da tabela, cria uma linha na tabela
  • th = table header, ou Cabeçalho da tabela, cria um cabeçalho na coluna
  • td = table data, ou dados da tabela, inseri dados dentro da celula de uma tabela
